Exploring India's COVID-19 Journey: A Case Study Analysis


CONCEPT

In this project, you will be working on real-world dataset to analyze India's Covid-19 journey. This project aims at cleaning the dataset, analyzing the given dataset and mine informational insights using Python and MySQL.

The project helps in understanding age-wise distribution, confirmed cases, recoveries, and daily decreases in the India's Covid-19 cases. Additionally, we will also examine state-wise data on cases, death and recovery status, available hospital beds, testing data, and population distribution. By uncovering insights from these datasets, this study aims to provide a comprehensive understanding of the COVID-19 situation in India, enabling informed decision-making and effective strategies to combat the pandemic.

TOOLS USED

Tools used in this project:

  • Python
  • MySQL
  • Tableau
